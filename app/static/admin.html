Document templates

DOCX templates use placeholders such as {{employee_name}}. PDF templates use positions in percentages from the top-left corner.
Input order in each card: DOCX placeholder, PDF page, PDF X %, PDF Y %, font size.
The PDF scrolls normally. To save a field position, enable placement mode and click inside the document.
